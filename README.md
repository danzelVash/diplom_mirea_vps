# Smart Home Control System

VPS-репозиторий дипломного проекта для системы управления внутренней инфраструктурой жилого помещения.

## Структура

- `services/` — сервисы online-контура на VPS.
- `docs/` — архитектурные описания и взаимодействие сервисов.
- `deploy/` — базовые файлы для локального развертывания.

## Сервисы

- `api-gateway` — внешняя точка входа.
- `edge-bridge-service` — внешний шлюз для интеграции с edge-репозиторием.
- `device-service` — устройства, состояния и команды.
- `context-service` — агрегированный контекст помещения.
- `scenario-service` — сценарии и принятие решений.
- `voice-service` — обработка голосовых команд.
- `vision-service` — видеоаналитика и идентификация.
- `notification-service` — уведомления и тревожные события.

## Режимы работы

- `online` — основной режим этого репозитория: вычисления, координация и принятие решений на VPS.
- `offline` — реализуется в отдельном edge-репозитории; здесь остаются только внешние контракты интеграции.
